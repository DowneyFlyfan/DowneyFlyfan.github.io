最优解问题

贪心算法

贪心算法的原理

每一步都做出在当前状态下最好的选择，以期最终能够得到全局最优解

实例

买卖股票的最佳时机 给定一个数组，它的第 i 个元素是一支给定股票第 i 天的价格。如果你最多只允许完成一笔交易（即买入和卖出一支股票），设计一个算法来计算你所能获取的最大利润。

注意你不能在买入股票前卖出股票。

示例 1:

输入: [7,1,5,3,6,4] 输出: 5 解释: 在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5。 注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格。

示例 2:

输入: [7,6,4,3,1] 输出: 0 解释: 在这种情况下, 没有交易完成, 所以最大利润为 0。

from typing import List

stock = []

n=int(input("一共有多少天的股票数据:"))

for i in range(n):
    num = input("请输入第" + str(i+1) + "天的股票:")
    stock.append(int(num))

def max_profit(prices: List[int]) -> int:
    # 最大利润初始化为0
    max_profit = 0
    # 最小价格初始化为第一个价格
    min_price = prices[0]

    # 遍历数组，找到最小价格和最大利润
    for price in prices:
        # 如果当前价格小于最小价格，更新最小价格
        if price < min_price:
            min_price = price
        # 计算当前价格与最小价格的差，即当前利润
        curr_profit = price - min_price
        # 如果当前利润大于最大利润，更新最大利润
        if curr_profit > max_profit:
            max_profit = curr_profit

    print("最大利润为:"+str(max_profit))
    return max_profit

max_profit(stock)

动态规划算法(Dynamic Progarmming, DP)

原理

将问题分解为若干个子问题并逐个求解子问题来求解最优化问题

性质

  1. 最优子结构：问题的最优解可以由子问题的最优解构成。

  2. 无后效性：子问题的解一旦确定，不受后续状态的影响。

  3. 重叠子问题：子问题之间存在重叠的情况，即多个子问题共享同一个子问题的解。

实例

def fibonacci(n):
	if n<=1:
		return n
	
	# 初始化dp数组
	dp = [0]*(n+1)
	dp[1] = 0

	# 填充dp数组
	for i in range(2,n+1):
		dp[i] = dp[i-1] + dp [i-2]
			
	return dp[n]

def knapsack(W, wt, val, n): # 背包容量，物品重量列表，物品价值列表，物品数量

    # 创建二维数组用于存储子问题的解
    K = [[0 for x in range(W+1)] for x in range(n+1)]

    # 构建K数组，计算每个子问题的最优解
    for i in range(n+1):
		for w in range(W, -1, -1):  # 逆序遍历可用容量，从 W 到 0
            # 处理边界情况，即没有物品或背包容量为0的情况下，最大价值都为0
            if i == 0 or w == 0:
                K[i][w] = 0
            # 如果当前物品的重量小于等于背包容量，那么有两种情况：
            # 1. 把当前物品装入背包，所得的总价值为当前物品价值加上前i-1个物品在剩余空间w-wt[i-1]下的最大价值K[i-1][w-wt[i-1]]
            # 2. 不把当前物品装入背包，所得的总价值为前i-1个物品在背包容量w下的最大价值K[i-1][w]
            # 取这两种情况的最大值，即为当前子问题的最优解
            elif wt[i-1] <= w:
                K[i][w] = max(val[i-1] + K[i-1][w-wt[i-1]], K[i-1][w])
            # 如果当前物品的重量大于背包容量，则只能不装入背包，所得的总价值为前i-1个物品在背包容量w下的最大价值K[i-1][w]
            else:
                K[i][w] = K[i-1][w]
    
    # 返回问题的最优解，即在背包容量为W的情况下，前n个物品能够获得的最大价值
    return K[n][W]

遗传算法(GA)

步骤

  1. 初始化种群：随机生成一组初始解作为种群，其中每个解表示问题的一个可行解。

  2. 适应度函数：定义一个适应度函数来评价每个解的优劣，根据适应度函数计算每个解的适应度值。

  3. 选择：根据适应度值选择一些优秀的解，将它们保留下来作为下一代种群的父代。

  4. 交叉：随机选择一些父代解，进行交叉操作，产生新的解作为下一代种群的子代。

  5. 变异：对一些子代解进行变异操作，产生一些新的解，加入到下一代种群中。

  6. 判断终止条件：检查是否满足终止条件，如果满足，则输出最优解，否则回到步骤3。

实例

import random

def knapsack_ga(W, wt, val, n, pop_size=50, elite_size=10, mutation_rate=0.01, generations=100):

'''
W: 背包最大承重
wt: 每个物品的重量，表示为一个长度为n的列表
val: 每个物品的价值，表示为一个长度为n的列表
n: 物品的数量
pop_size: 种群大小，默认为50
elite_size: 精英数量，默认为10
mutation_rate: 变异概率，默认为0.01
generations: 进化代数，默认为100
'''

    # 初始化种群
    population = []
    for i in range(pop_size):
        individual = [random.randint(0, 1) for _ in range(n)]
        population.append(individual)

    # 适应度函数
    def fitness(individual):
        return sum(val[i] for i in range(n) if individual[i] == 1)

    # 选择
    def selection(population):
        return random.sample(population, elite_size) + [max(population, key=fitness) for _ in range(pop_size-elite_size)]

    # 交叉
    def crossover(ind1, ind2):
        k = random.randint(1, n-1)
        return ind1[:k] + ind2[k:], ind2[:k] + ind1[k:]

    # 变异
    def mutation(individual):
        for i in range(n):
            if random.random() < mutation_rate:
                individual[i] = 1 - individual[i]

    # 进化
    for generation in range(generations):
        # 选择
        selected_population = selection(population)

        # 交叉
        offspring_population = []
        for i in range(pop_size):
            parent1, parent2 = random.sample(selected_population, 2)
            offspring1, offspring2 = crossover(parent1, parent2)
            offspring_population.append(offspring1)
            offspring_population.append(offspring2)

        # 变异
        for individual in offspring_population:
            mutation(individual)

        # 更新种群
        population = offspring_population

    # 返回最优解
    return max(population, key=fitness)

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        # 每次遍历将最大值移动到末尾
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                # 交换相邻元素的位置
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr

快速排序

原理

选择一个基准元素，通过一趟排序将待排序的序列分成两个部分，其中一个部分的所有元素都比基准元素小，另一个部分的所有元素都比基准元素大，然后对这两个部分分别进行快速排序，最终将整个序列排序。

def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    else:
        pivot = arr[0]  # 选择第一个元素作为基准元素
        left = [x for x in arr[1:] if x < pivot]  # 将小于基准元素的放到左边
        right = [x for x in arr[1:] if x >= pivot]  # 将大于等于基准元素的放到右边
        return quick_sort(left) + [pivot] + quick_sort(right)  # 递归调用左右两边排序后的结果

归并排序

原理

  • 分解：将待排序的序列递归地分成两个子序列，直到每个子序列只有一个元素为止

  • 合并：将两个已排好序的子序列合并成一个有序序列

  • 合并所有子序列，得到最终的有序序列

def merge_sort(arr):
    if len(arr) > 1:
        # 分解：将 arr 分成两个子序列 left 和 right
        mid = len(arr) // 2 # //是整除符号
        left = arr[:mid]
        right = arr[mid:]

        # 对左右子序列分别进行排序
        merge_sort(left)
        merge_sort(right)

        # 合并：将已排好序的 left 和 right 合并成一个有序序列
        i = j = k = 0

        while i < len(left) and j < len(right):
            if left[i] < right[j]:
                arr[k] = left[i]
                i += 1
            else:
                arr[k] = right[j]
                j += 1
            k += 1

        while i < len(left):
            arr[k] = left[i]
            i += 1
            k += 1

        while j < len(right):
            arr[k] = right[j]
            j += 1
            k += 1
    return arr

def merge(arr, left, mid, right):
    temp = []
    i, j = left, mid + 1
    while i <= mid and j <= right:
        if arr[i] <= arr[j]:
            temp.append(arr[i])
            i += 1
        else:
            temp.append(arr[j])
            j += 1
    while i <= mid:
        temp.append(arr[i])
        i += 1
    while j <= right:
        temp.append(arr[j])
        j += 1
    arr[left:right + 1] = temp

def merge_sort(arr):
    n = len(arr)
    size = 1
    while size < n:
        left = 0
        while left < n:
            mid = left + size - 1
            if mid >= n - 1:
                break
            right = min(mid + size, n - 1)
            merge(arr, left, mid, right)
            left = right + 1
        size *= 2
    return arr
