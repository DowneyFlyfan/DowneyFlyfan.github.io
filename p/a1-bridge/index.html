  1. Input Impedance
  • General Case

$$ \begin{gather} \begin{aligned} Z_{in}(l) &= \frac{ V(l) }{ I(l) } \\ &= \frac{ V_{(+)} e^{ -\gamma x} + V_{(-)}e^{ \gamma x}}{ I_{(+)} e^{- \gamma x} + I_{(-)} e^{ \gamma x}} \\ &= Z_0 \frac{ 1 + \Gamma_L e^{-2 \gamma l} }{ 1 - \Gamma_L e^{-2 \gamma l}}\\ &= Z_0 \frac{ Z_L + Z_0 tanh( \gamma l) }{ Z_0 + Z_L tanh( \gamma l) } \end{aligned} \end{gather} $$

Telegrapher’s equation

  1. Line Voltage - Current Relation

$$ \begin{gather} \begin{aligned} \frac{ \partial V(x) }{ \partial x } = - (R + j \omega L) I(x) \\ \frac{ \partial I(x) }{ \partial x } = - (G + j \omega C) V(x) \end{aligned} \end{gather} $$

  1. Propogation Constant in General Case

$$ \begin{aligned} \frac{ \partial^2 V }{ \partial x^2 } &= (R + j \omega L )(G + j \omega C) V \end{aligned} $$

  • Suppose $V = V_0 e^{- \gamma x}$

$$ \begin{gather} \begin{aligned} \gamma &= \sqrt{(R + j \omega L)(G + j \omega C)} \\ \mathbb { Re } ( \gamma) &= \alpha = (a^2 + b^2)^{1/4} cos( \phi) \\ \mathbb { Im }( \gamma) &= \beta = (a^2 + b^2)^{1/4}sin( \phi) \\ a &= RG - \omega^2 LC = \omega^2 LC \Big[ ( \frac{ R }{ \omega L}) ( \frac{ G }{ \omega C }) - 1) \Big] \\ b &= \omega C R + \omega L G = \omega^2 LC ( \frac{ R }{ \omega L } + \frac{ G }{ \omega C }) \\ \phi &= tan^{-1}( \frac{ b }{ a }) \\ L, C, \omega &\neq 0 \end{aligned} \end{gather} $$

$$ \begin{gather} \begin{aligned} \frac{ \partial^2 V }{ \partial x^2 } &= \gamma^2 V \\ \frac{ \partial^2 I }{ \partial x^2 } &= \gamma^2 I \end{aligned} \end{gather} $$

  1. Propogation Constant in Idealized Case

$$ \begin{aligned} \frac{ R }{ \omega L} \ll 1, \frac{ G }{ \omega C } \ll 1 \end{aligned} $$

$$ \begin{gather} \begin{aligned} \mathbb { Re } ( \gamma) & \approx \frac{ 1 }{ 2 } \sqrt{ LC } \Big( \frac{ R }{ L } + \frac{ G }{ C } \Big) \\ \mathbb { Im } ( \gamma) & \approx \omega \sqrt{ LC } \end{aligned} \end{gather} $$

$$ \begin{gather} \begin{aligned} V_{out}(x,t) &\approx V_{in}(t - \sqrt{ LC } x) e^{ - \dfrac{ 1 }{ 2 } \sqrt{ LC } \Big( \dfrac{ R }{ L } + \dfrac{ G }{ C } \Big)x} \end{aligned} \end{gather} $$

$$ \begin{gather} \begin{aligned} \frac{ \partial^2 V }{ \partial x^2 } &= - \omega^2 LC \cdot V \\ \frac{ \partial^2 I }{ \partial x^2 } &= - \omega^2 LC \cdot I \\ \end{aligned} \end{gather} $$
