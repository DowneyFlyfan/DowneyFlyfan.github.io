Impedance & Reflected Coefficient

Characteristic Impedance

$$ \begin{gather} \begin{aligned} Z_0 &= \frac{ V_{(+)} (l) }{ I_{(+)} (l) } \\ &= \sqrt{ \frac{ R + j \omega L }{ G + j \omega C } } \\ &= \frac{ |V_{max}| }{ |I_{max}| } \\ &= \frac{ |V_{min}| }{ |I_{min}| } \end{aligned} \end{gather} $$

Reflected Coefficient

Reflected Coefficient Formula

$$ \begin{gather} \begin{aligned} \Gamma (x’) &= \frac{ V^-(x’) }{ V^+ (x’) } = - \frac{ I^- (x’) }{ I^+ (x’) } \\ &= \frac{ V_2 ^- e^{ -j \beta x’} }{ V_2 ^+ e^{ j \beta x’}} \\ &= \Gamma_L e^{-j 2 \beta x’} \\ &= |\Gamma_L| e^{j ( \phi_L - 2 \beta x’)} \end{aligned} \end{gather} $$

Reflected Coefficient at Terminal

$$ \begin{gather} \begin{aligned} \Gamma_L &= \frac{ V_2^- }{ V_2^+ } \\ &= \frac{ V_2 - I_2 Z_0 }{ V_2 + I_2 Z_0 } \\ &= \frac{ Z_L - Z_0}{ Z_L + Z_0 } \\ &= | \Gamma_L | e^{j \phi_L} \end{aligned} \end{gather} $$

VSWR (Voltage Standing Wave Ratio) & Traveling Wave Coefficient

$$ \begin{gather} \begin{aligned} VSWR &= \rho = \frac{ |V_{max} }{ |V_{min}| } \\ &= \frac{ 1 + | \Gamma_L| }{ 1 - | \Gamma_L| } \\ K &= \frac{ 1 }{ \rho} = \frac{ |V_{min}| }{ |V_{max}|} \\ &= \frac{ 1 -| \Gamma_L| }{ 1 + | \Gamma_L | } \end{aligned} \end{gather} $$

Input Impedance

$$ \begin{gather} \begin{aligned} Z_{in}(x’) &= \frac{ V(x’) }{ I(x’) } \\ &= Z_0 \frac{ 1 + \Gamma (x’) e^{-j 2 \beta x’}}{1 - \Gamma (x’) e^{-j 2 \beta x’}} \\ &= Z_0 \frac{ Z_L + j Z_0 tan( \beta x’) }{ Z_L - j Z_0 tan( \beta x’) } \end{aligned} \end{gather} $$

$$ \begin{gather} \begin{aligned} Z_{in}(0) &= Z_0 \frac{ 1 + \Gamma_L }{ 1 - \Gamma_L } = Z_L \\ Z_{in}(l) &= Z_0 \end{aligned} \end{gather} $$

Telegrapher’s equation

Telegrapher’s Equation

$$ \begin{gather} \begin{aligned} \frac{ \partial V(x,t) }{ \partial x } = - (R + j \omega L) I(x) \\ \frac{ \partial I(x,t) }{ \partial x } = - (G + j \omega C) V(x) \end{aligned} \end{gather} $$

Wave Equation for a homogeneous Transmission Line

  1. Equation

$$ \begin{gather} \begin{aligned} \frac{ \partial^2 V }{ \partial x^2 } - \gamma^2 V &= 0 \\ \frac{ \partial^2 I }{ \partial x^2 } - \gamma^2 I &= 0 \\ \gamma &= \sqrt{ (R + j \omega L)(G + j \omega C) } = \alpha + j \beta \\ \mathbb { Re } ( \gamma) &= \alpha = (a^2 + b^2)^{1/4} cos( \phi) \\ \mathbb { Im }( \gamma) &= \beta = (a^2 + b^2)^{1/4}sin( \phi) \\ a &= RG - \omega^2 LC = \omega^2 LC \Big[ ( \frac{ R }{ \omega L}) ( \frac{ G }{ \omega C }) - 1) \Big] \\ b &= \omega C R + \omega L G = \omega^2 LC ( \frac{ R }{ \omega L } + \frac{ G }{ \omega C }) \\ \phi &= tan^{-1}( \frac{ b }{ a }) \\ L, C, \omega &\neq 0 \end{aligned} \end{gather} $$

  1. Solution in General Case

$$ \begin{gather} \begin{aligned} V(x) &= A_1 e^{- \gamma x} + A_2 e^{ \gamma x} \\ I(x) &= \frac{ 1 }{ Z_0 }(A_1 e^{- \gamma x} - A_2 e^{ \gamma x}) \\ Z_0 &= \sqrt{ \dfrac{ R + j \omega L }{ G + j \omega C } } \end{aligned} \end{gather} $$
