Maxwell Equation

Differetial Form

$$ \begin{gather} \begin{aligned} \nabla \times \overset{ \longrightarrow }{ \varepsilon} &= - \frac{ \partial \pmb{ B } }{ \partial t } - \pmb{ M } \\ \nabla \times \overset{ \longrightarrow }{ H} &= \frac{ \partial \pmb{ D } }{ \partial t } + \pmb{ J } \\ \nabla \cdot \overset{ \longrightarrow }{ \varepsilon} &= \rho \\ \nabla \cdot \overset{ \longrightarrow }{ H} &= 0 \end{aligned} \end{gather} $$

Integral Form

$$ \begin{gather} \begin{aligned} \oint_{ }^{ } \overrightarrow { \varepsilon} d \overrightarrow{l} &= - \frac{ \partial }{ \partial t } \int_{ }^{ } \pmb{ B } d \overrightarrow{S} - \int_{ }^{ } \pmb{ M } d \overrightarrow{S} \\ \oint_{ }^{ } \overrightarrow { H} d \overrightarrow{l} &= \frac{ \partial }{ \partial t } \int_{ }^{ } \pmb{ D } d \overrightarrow{S} + I \\ \int_{ }^{ } \overrightarrow{ \varepsilon } d \overrightarrow{ S } &= \int_{ }^{ } \rho dV \\ \int_{ }^{ } \overrightarrow{ H} d \overrightarrow{ S } &= 0 \end{aligned} \end{gather} $$
