def knapsack(weights, values, capacity):
    n = len(weights)
    # dp[i][j]: Âú®Ââç i ‰∏™Áâ©ÂìÅ‰∏≠ÔºåËÉåÂåÖÂÆπÈáè‰∏∫ j Êó∂ÁöÑÊúÄÂ§ß‰ª∑ÂÄº
    dp = [[0] * (capacity + 1) for _ in range(n + 1)] 

    for i in range(1, n + 1):
        for j in range(1, capacity + 1):
            if weights[i - 1] <= j:
                dp[i][j] = max(dp[i - 1][j], values[i - 1] + dp[i - 1][j - weights[i - 1]])
            else:
                dp[i][j] = dp[i - 1][j]

    selected_items = []
    i, j = n, capacity
    while i > 0 and j > 0:
        if dp[i][j] != dp[i - 1][j]:
            selected_items.append(i)
            j -= weights[i - 1]
        i -= 1

    selected_items.reverse()
    return dp[n][capacity], selected_items

Âü∫Â∞ºÊåáÊï∞Ë∂äÂ∞èË°®Á§∫Êï∞ÊçÆÈõÜË∂äÁ∫Ø„ÄÇ

ÂØπ‰∫é‰∏Ä‰∏™ÁâπÂæÅ $A$ ÂíåÂÖ∂ÂèñÂÄº $a_i$ÔºåÂÅáËÆæ $D$ Ë°®Á§∫ÂΩìÂâçËäÇÁÇπÁöÑÊï∞ÊçÆÈõÜÔºå$D_i$ Ë°®Á§∫ÁâπÂæÅ $A$ ÂèñÂÄº‰∏∫ $a_i$ Êó∂ÁöÑÂ≠êÊï∞ÊçÆÈõÜÔºåÂàôÂü∫Â∞ºÊåáÊï∞ËÆ°ÁÆóÂÖ¨Âºè‰∏∫Ôºö

$$Gini(D,A)=\sum_{i=1}^{k}\frac{|D_i|}{|D|}Gini(D_i)$$

ÂÖ∂‰∏≠Ôºå$k$ Ë°®Á§∫ÁâπÂæÅ $A$ ÁöÑÂèñÂÄºÊï∞ÁõÆÔºå$|D_i|$ Ë°®Á§∫Â≠êÊï∞ÊçÆÈõÜ $D_i$ ‰∏≠Ê†∑Êú¨ÁöÑÊï∞ÁõÆÔºå$|D|$ Ë°®Á§∫ÂΩìÂâçËäÇÁÇπÁöÑÊ†∑Êú¨Êï∞ÁõÆÔºå$Gini(D_i)$ Ë°®Á§∫Â≠êÊï∞ÊçÆÈõÜ $D_i$ ÁöÑÂü∫Â∞ºÊåáÊï∞ÔºåËÆ°ÁÆóÂÖ¨Âºè‰∏∫Ôºö

$$Gini(D_i)=1-\sum_{j=1}^{c}p_{ij}^2$$

ÂÖ∂‰∏≠Ôºå$c$ Ë°®Á§∫Á±ªÂà´Êï∞ÁõÆÔºå$p_{ij}$ Ë°®Á§∫Â≠êÊï∞ÊçÆÈõÜ $D_i$ ‰∏≠Â±û‰∫éÁ¨¨ $j$ Á±ªÁöÑÊ†∑Êú¨Âç†ÊØî„ÄÇ

Âü∫Â∞ºÊåáÊï∞Ë∂äÂ∞èË°®Á§∫‰ΩøÁî®ÁâπÂæÅ $A$ ËøõË°åÂàíÂàÜÂêéÔºåÂæóÂà∞ÁöÑÂ≠êÊï∞ÊçÆÈõÜÁöÑÁ∫ØÂ∫¶Ë∂äÈ´ò„ÄÇÂõ†Ê≠§ÔºåÂÜ≥Á≠ñÊ†ëÁÆóÊ≥ïÈÄâÊã©Âü∫Â∞ºÊåáÊï∞ÊúÄÂ∞èÁöÑÁâπÂæÅ‰Ωú‰∏∫ÊúÄ‰ºòÁâπÂæÅËøõË°åÂàíÂàÜ„ÄÇ

  • ËÆ°ÁÆóÁâπÂæÅÈòàÂÄº(Áî®‰ø°ÊÅØÂ¢ûÁõäË°®Á§∫ÁâπÂæÅÈòàÂÄº)

‰ø°ÊÅØÂ¢ûÁõäÊòØËØÑ‰ª∑ÁâπÂæÅÈáçË¶ÅÊÄßÁöÑ‰∏ÄÁßçÊåáÊ†áÔºåÂÆÉË°®Á§∫ÁâπÂæÅ $x_j$ ÂØπÂàÜÁ±ªÁöÑË¥°ÁåÆÁ®ãÂ∫¶„ÄÇ‰ø°ÊÅØÂ¢ûÁõäË∂äÂ§ßÔºåËØ¥ÊòéÁâπÂæÅÁöÑÈáçË¶ÅÊÄßË∂äÈ´ò„ÄÇÂèØ‰ª•ÈÄöËøá‰ª•‰∏ãÂÖ¨ÂºèËÆ°ÁÆó‰ø°ÊÅØÂ¢ûÁõäÔºö

$$ \begin{aligned} Gain(D,x_j) &= H(D) - \sum_{i=1}^m \frac{|D_i|}{|D|} H(D_i) \\ H &= -\sum p(x) \times log_2[p(x)] \end{aligned} $$

  • ÂÖ∂‰∏≠ $H(D)$ Ë°®Á§∫Êï∞ÊçÆÈõÜ $D$ ÁöÑÁÜµÔºå$m$ Ë°®Á§∫ÁâπÂæÅ $x_j$ÁöÑÂèØËÉΩÂèñÂÄº‰∏™Êï∞Ôºå$D_i$ Ë°®Á§∫Êï∞ÊçÆÈõÜ $D$ Âú®ÁâπÂæÅ $x_j$ ÂèñÂÄº‰∏∫ $i$ Êó∂ÂØπÂ∫îÁöÑÂ≠êÈõÜÔºå$|D_i|$ Ë°®Á§∫Â≠êÈõÜ $D_i$ ÁöÑÂ§ßÂ∞èÔºå$|D|$ Ë°®Á§∫Êï∞ÊçÆÈõÜ $D$ ÁöÑÂ§ßÂ∞è„ÄÇ

  1. ÂØπÁâπÂæÅ $x_j$ ÁöÑÂèñÂÄºËøõË°åÊéíÂ∫èÔºåÂæóÂà∞‰∏Ä‰∏™ÊúâÂ∫èÁöÑÂàóË°® $[x_1,x_2,\cdots,x_m]$

  2. ÂØπ‰∫éÁõ∏ÈÇªÁöÑÂèñÂÄº $x_i$ Âíå $x_{i+1}$ÔºåÂ∞ÜÂÆÉ‰ª¨‰Ωú‰∏∫ÁâπÂæÅÈòàÂÄº $s$ÔºåÂ∞ÜÊï∞ÊçÆÈõÜÂàíÂàÜ‰∏∫‰∏§‰∏™Â≠êÈõÜ $D_1={x\in D|x_j\leq s}$ Âíå $D_2={x\in D|x_j>s}$

  3. ËÆ°ÁÆó‰ΩøÁî®ÁâπÂæÅÈòàÂÄº $s$ ËøõË°åÂàíÂàÜÁöÑ‰ø°ÊÅØÂ¢ûÁõä $Gain(D,x_j,s)$

$$Gain(D,x_j,s) = H(D) - \frac{|D_1|}{|D|} H(D_1) - \frac{|D_2|}{|D|} H(D_2)$$
